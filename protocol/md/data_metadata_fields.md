# SLR Significant Data and Metadata Fields (SD & SMD)

| ID | TYPE | FEATURE | DESCRIPTION |
|:---:|:---:|:---:|:---|
| **SD-01** | Data | *Title* | Given document title. |
| **SD-02** | Data | *Abstract* | Reported document summary. |
| **SD-03** | Data | *Keywords* | Keywords or labels from the academic source. |
| **SD-04** | Data | *Author* | Listed contributors and collaborators. |
| **SD-05** | Data | *Methodology* | Reported research methods. |
| **SD-06** | Data | *Tools* | Used software or technologies. |
| **SD-07** | Data | *Results* | Main findings and contributions. |
| **SD-08** | Data | *Limitations* | Discussed limitations and challenges. |
| **SD-09** | Data | *Future Directions* | Proposed future work. |
| **SD-10** | Data | *References* | Cited works details and count. |
| **SMD-01** | Metadata | *Source* | Academic database name. |
| **SMD-02** | Metadata | *Venue* | Publication title. |
| **SMD-03** | Metadata | *Date Added* | Date added to the database. |
| **SMD-04** | Metadata | *Year* | Publication year. |
| **SMD-05** | Metadata | *DOI* | Digital Object Identifier. |
| **SMD-06** | Metadata | *Type* | Document type. |
