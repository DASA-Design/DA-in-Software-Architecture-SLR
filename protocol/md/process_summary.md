# SLR Process Task Summary

| ID | NAME | DESCRIPTION |
|:---:|:---:|:---|
| **T-001** | *Define research questions* | To postulate the research questions based on the preliminary search string. |
| **T-002** | *Specify search strategy* | To define the search strategy, including sources, time periods, and document types. |
| **T-003** | *Create search string* | To create the search string using the PICO strategy. |
| **T-004** | *Define inclusion/exclusion criteria* | To establish the inclusion and exclusion criteria. |
| **T-005** | *Review protocol quality* | To evaluate the protocol quality using control instruments. |
| **T-006** | *Update protocol* | To adjust the SLR protocol if necessary. |
| **T-007** | *Use search string* | To execute the search string in academic databases. |
| **T-008** | *Collect documents* | To collect documents and metadata from the search results. |
| **T-009** | *Apply inclusion/exclusion criteria* | To apply the inclusion and exclusion criteria to filter documents. |
| **T-010** | *Extract data and metadata* | To extract relevant data/metadata from documents. |
| **T-011** | *Update search string and strategy* | To refine the search string and strategy if necessary. |
| **T-012** | *Synthetize document data* | To summarize document sections (methodology, tools, results, insights, limitations, and future work). |
| **T-013** | *Perform snowballing references* | To expand the corpus through reference snowballing. |
| **T-014** | *Create document category map* | To create a keyword mapping scheme from document content. |
| **T-015** | *Classify documents* | To classify and map documents using the keyword mapping scheme. |
| **T-016** | *Summarize findings* | To summarize insights (contributions, trends, patterns, and knowledge gaps). |
| **T-017** | *Review report quality* | To assess the SLR report quality using control instruments. |
| **T-018** | *Update report* | To update the SLR report if necessary. |
| **T-019** | *Deliver report* | To deliver the final SLR report with diagrams and findings. |
