# SLR Data Objects Summary

| ID | NAME | DESCRIPTION |
|:---:|:---:|:---|
| **O-01** | *Research Question* | Defined research questions for dimensional analysis. |
| **O-02** | *Data Sources & Strategy* | Selected academic sources and search strategy in the review protocol. |
| **O-03** | *Search String* | Specified search string based on research questions. |
| **O-04** | *Inclusion/Exclusion Criteria* | Checklist of inclusion and exclusion criteria. |
| **O-05** | *Approved Protocol* | Complete and reviewed SLR protocol. |
| **O-06** | *Found Documents* | Documents found using search string, strategy, and sources. |
| **O-07** | *Primary Studies* | Primary documents after applying criteria. |
| **O-08** | *Tabulated Data* | Processed data and statistics from primary documents. |
| **O-09** | *Classification Mapping Scheme* | Classification mapping scheme for primary documents using analysis tools. |
| **O-10** | *Systematic Map* | Data and metadata charts from primary documents. |
| **O-11** | *Insights* | Analysis of primary documents (contributions, gaps, and challenges). |
| **O-12** | *Survey Report* | Definitive report for the Systematic Literature Review. |
