# DATA PROFILE

Comprehensive profile of the data collected, processed, and analyzed during the Systematic Literature Review (SLR) on Dimensional Analysis (DA) in Software Architecture. The data is organized across multiple stages of the review process, from initial bibliographic searches to final corpus analysis and visualization.

## Overview

The [data/](.) folder contains all raw data, intermediate processing outputs, and analytical results generated throughout the SLR process. The data is organized into four main categories:

1. **[Bibliography](bibliography/):** Raw search results from academic databases.
2. **[Zotero](zotero/):** Curated primary studies and metadata.
3. **[Voyant](voyant/):** Text analytics and visualization outputs.
4. **[Scripts](scripts/):** Data processing utilities and stopword analysis.

## Data Flow Summary

The data flows through the SLR process as follows:

```
1. SEARCH PHASE
   └─> bibliography/query/ (Raw BibTeX/RIS exports)
   └─> bibliography/export/ (CSV conversions)

2. SCREENING PHASE
   └─> Import into Zotero
   └─> Apply inclusion/exclusion criteria
   └─> Quality assessment

3. CURATION PHASE
   └─> zotero/primary-studies.csv (Final corpus)

4. ANALYSIS PHASE
   └─> Extract full-text from primary studies
   └─> Import into Voyant Tools
   └─> Generate analytics: voyant/data/
   └─> Generate visualizations: voyant/img/
   └─> Export formatted results: voyant/md/

5. SYNTHESIS PHASE
   └─> Use stopword analysis: scripts/
   └─> Refine term extraction
   └─> Answer research questions
```

## Data Quality & Reproducibility

### Completeness
- ✅ All five academic sources included
- ✅ Dual time windows (2010-2020, 2010-2024)
- ✅ Original formats preserved (BibTeX, RIS, CSV)
- ✅ Export formats standardized (CSV)

### Traceability
- Each file follows a consistent naming convention: `{number} - {source} - SLR DA Results {timeframe}.{format}`
- Zotero Key field links primary studies to original database records
- DOIs and URLs enable direct access to original publications

### Validation
- Corpus statistics verified: 6 documents, 56,296 words, 7,322 unique forms
- Cross-reference between Zotero exports and Voyant analysis confirms data integrity

### Reproducibility
To reproduce the data collection and analysis:
1. Execute search strings (see [`protocol/md/search_string.md`](../protocol/md/search_string.md)) in each database
2. Export results using specified formats to [`bibliography/query/`](bibliography/query/)
3. Convert to CSV and save to [`bibliography/export/`](bibliography/export/)
4. Import into Zotero and apply screening criteria
5. Export selected studies to [`zotero/primary-studies.csv`](zotero/primary-studies.csv)
6. Upload full-text PDFs to Voyant Tools
7. Run stopword analysis: `python scripts/stopword-analitics.py`
8. Generate Voyant visualizations and export to [`voyant/`](voyant/) subfolders
